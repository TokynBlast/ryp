## What Is Ryp

Ryp is a text editor, all within the terminal!<br>
It's lightweight, and feature rich!<br>
Including things, like a git viewer, search, replace, and so much more.

## Features
There are multiple features, such as...
- git
- search
- replace
- replace all
- customizability
- file tree
- text editing
- syntax highlighting

And many more planned features, such as Lua plugins, config files, resource limiting, and much more!
